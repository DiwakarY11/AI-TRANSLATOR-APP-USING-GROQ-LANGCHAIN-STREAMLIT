* **Lightning Fast:** Powered by Groq's LPU (Language Processing Unit) for near-instant translation results.

* **Clean UI:** Simple, intuitive interface built with Streamlit.
* **Multi-language Support:** Translate from English into Hindi, French, Spanish, German, and more.
